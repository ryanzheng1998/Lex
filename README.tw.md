# Lexical Analyzer Generator

> 用 JavaScript（Node.js） + Ramda 實作 Lexical Analyzer Generator

## 開始使用它

### 安裝

```bash
npm install
```

### 在開發環境中編譯並執行

```bash
npm run build
```

## 完成事項

- [x] Regex -> Nondeterministic Finite Autometa（NFA）
- [x] NFA -> Deterministic Finite Autometa（DFA）
- [ ] DFA -> Optimized DFA
- [ ] Optimized DFA -> Hash Table Array
